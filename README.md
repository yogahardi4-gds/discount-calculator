## Discount Calculator with Python
A simple Python program to calculate the final price after applying a discount. This tool is perfect for small businesses, online sellers, or anyone who wants a quick way to calculate discounts on products or services.

## Features
- Calculate final price after discount
- Validate discount input (0-100%)
- Run directly from terminal or command prompt
- Simple and user-friendly interface

## How to Use
1. Clone this repository:
```bash
git clone https://github.com/yogahardi4-gds/discount-calculator.git
```
2. Navigate to the project folder:
```bash
cd discount-calculator
```
3. Run the program:
```bash
python discount_calculator.py
```
4. Follow the prompts to enter:
- Original price
- Discount percentage
5. The program will display the final price after discount.

## Technologies / Requirements
- Python 3.x
- No external libraries required
